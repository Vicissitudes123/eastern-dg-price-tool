# 东航危险品运价对比工具 - 设计构思

## 项目背景
这是一个内部业务工具，用于查看东航危险品运价的跨年趋势对比，帮助验证代理报价的合理性。核心用户是公司内部的物流/运价分析人员。

## 方案一：Bloomberg Terminal 风格 - 数据密集型专业界面

**设计运动**: 金融终端 / 数据仪表盘美学
**核心原则**:
1. 数据密度优先 - 在有限空间内展示最多有效信息
2. 深色背景减少视觉疲劳 - 适合长时间数据分析
3. 高对比度色彩编码 - 涨跌一目了然
4. 零装饰 - 每个像素都服务于数据

**色彩哲学**: 深蓝黑底色(#0a0e17)，亮绿(#00d97e)表示下降/有利，亮红(#ff6b6b)表示上涨/不利，冰蓝(#4cc9f0)作为中性信息色。这套配色模拟金融终端，让用户立即进入"分析模式"。

**布局范式**: 左侧窄栏为筛选控制面板（航线、年份、月份），右侧大面积为图表+数据表格的上下分割布局。

**标志性元素**: 
1. 实时闪烁的数据点标记当前最新运价
2. 涨跌箭头+百分比标签直接叠加在折线图上

**交互哲学**: 键盘优先，快速筛选，hover显示详细数据tooltip

**动画**: 数据加载时数字滚动效果，图表绘制时的逐点描绘动画

**字体系统**: JetBrains Mono用于数字/代码，Noto Sans SC用于中文标签

## 方案二：航空地图导航风格 - 空间化数据探索

**设计运动**: 地理信息可视化 / 航空导航美学
**核心原则**:
1. 以地图为中心的空间导航 - 航线即路径
2. 层级递进的信息揭示 - 从全局到细节
3. 航空工业的精密感 - 仪表盘质感
4. 蓝天白云的明亮基调

**色彩哲学**: 天空蓝渐变背景，白色卡片浮层，深海蓝(#1a365d)作为主色，金色(#d69e2e)作为高亮/警告色。模拟飞行仪表盘的冷静专业感。

**布局范式**: 顶部为世界地图展示航线，点击航线后下方展开详细价格对比面板，类似Google Flights的交互模式。

**标志性元素**:
1. 航线弧线连接PVG到各目的地
2. 目的地气泡大小反映价格变化幅度

**交互哲学**: 点击探索，视觉引导，从宏观到微观

**动画**: 航线飞行轨迹动画，气泡脉冲效果

**字体系统**: DM Sans用于标题，Source Sans Pro用于正文，Tabular Nums用于数字

## 方案三：清晰工具型界面 - 高效数据分析工作台

**设计运动**: 现代SaaS工具 / Notion-like 简洁功能主义
**核心原则**:
1. 功能即设计 - 界面服务于操作效率
2. 清晰的信息层级 - 筛选→图表→数据→偏差
3. 柔和但不失专业的色调 - 适合日常办公环境
4. 响应式布局适配不同屏幕

**色彩哲学**: 暖白底色(#fafaf9)，石墨灰文字(#1c1917)，三条年份折线分别用靛蓝(#4f46e5)、翠绿(#059669)、琥珀(#d97706)区分2024/2025/2026年。红色(#dc2626)标记不利偏差，绿色(#16a34a)标记有利偏差。色彩选择基于：三条线需要在同一图表中清晰区分，同时涨跌提示需要直觉化。

**布局范式**: 顶部固定工具栏（航线选择+年份/月份筛选器），中部大面积折线图区域，底部数据明细表格带偏差列。左右不分栏，采用全宽流式布局最大化图表展示面积。

**标志性元素**:
1. 筛选器采用标签式切片器，可多选年份和月份
2. 偏差列用色块+箭头+百分比三重编码

**交互哲学**: 所见即所得的筛选，改变任何筛选条件图表立即响应更新

**动画**: 筛选切换时图表平滑过渡动画，数据点hover时的弹性放大效果

**字体系统**: Inter用于界面元素，Noto Sans SC用于中文，Tabular数字对齐

## 选定方案：方案三 - 清晰工具型界面

选择理由：这是一个内部业务工具，经理的核心需求是"好筛选"、"数值要明显"、"折线图对比"。方案三最直接地服务于这些需求，不会因为过度设计而分散注意力。同时暖白色调适合日常办公使用，三色年份区分方案清晰直观。
